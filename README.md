# Smart Document Search

An AI-powered document retrieval and question-answering system that combines OCR, semantic search, and AI-generated responses.

## Features

- OCR-based text extraction
- Semantic and keyword search
- Multi-format document support
- AI-powered question answering
- User authentication
- Search history management

## Technologies Used

### Backend
- FastAPI
- Python
- SQLAlchemy

### Database
- SQLite

### AI & Search
- Google Gemini API
- FAISS
- Sentence Transformers
